二：http://solveme.safflower.kr/prob/name_check

这道题十分有意思，我其实想了很久，突然突发奇想才想出来的

代码十分有趣，按照代码来，我们需要select 1,1,1,1；四个1出来才能满足结果，也即下面代码每一行为真

第一行意思为你第一个必须以a开头

第二行就是你输入的需要有d

第三行就是输入的是xxmxx（xx为任意字符，中间是字符m）

第四行就是字符以in结尾

这摆明着要叫我们输入admin啊，可惜前面的正则残酷的将其过滤。

查看了文档，想过很多种方法，今天突发奇想 ||，这在sqlite中是一个字符连接的，代码的正则中没有将其过滤，也没有过滤’,所以最后就是http://solveme.safflower.kr/prob/name_check/?name=ad%27||%27min

出来结果。这个网站题目有点意思。再接再厉（简单的题目就不说了）
